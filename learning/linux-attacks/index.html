🧠 Linux Attack Lab (Based on Udemy Course)

This slide was created by reorganizing and summarizing content from the Udemy course “Linux攻撃実践演習” as a learning output.

The exercise involves setting up the VulnHub machine Stapler in a virtual environment and walking through reconnaissance, scanning, and exploitation steps.

📋 Overview

  • Target: Stapler (Beginner to Intermediate)
  • Tools Used: Netdiscover, nmap, Searchsploit, Metasploit
  • Attack Vector: Samba 4.3 - is_known_pipename vulnerability
  • Goal: Retrieve flag.txt

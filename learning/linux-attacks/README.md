# 🧠 Linux攻撃演習（Udemyベース）

このスライドは、Udemy講座『Linux攻撃実践演習』をベースに、内容を再構成し、学習アウトプットとしてまとめたものです。

## 📋 概要

- 対象：Stapler（難易度：初〜中級）
- 使用ツール：Netdiscover, nmap, Searchsploit, Metasploit
- 攻撃ベクトル：Samba 4.3 の `is_known_pipename` 脆弱性
- ゴール：`flag.txt` の取得まで
